# Portfolio Roadmap V2

## Milestone 01: Portfolio Control Plane

Objective: Establish Meta-Agent as the portfolio governance and routing authority.

Deliverables: Portfolio registry, authority model, owner decision log, approval routing, evidence model.

Dependencies: Canonical repository inventory.

Exit Criteria: Registry and governance docs validate.

## Milestone 02: AgentOps Runtime Foundation

Objective: Establish deterministic runtime contracts for governed tasks.

Deliverables: Task schemas, runtime evidence schemas, initial runtime docs, skill registry validation.

Dependencies: Meta-Agent task contracts.

Exit Criteria: Runtime schemas and skills validate.

## Milestone 03: Governed Dry-Run Runtime

Objective: Implement local dry-run runtime behavior without live execution.

Deliverables: Intake, routing, approvals, traces, no-op executor, sandbox simulation.

Dependencies: Runtime foundation and approval policy.

Exit Criteria: Runtime tests pass and live execution remains disabled.

## Milestone 04: Portfolio Integration

Objective: Connect Meta-Agent, AgentOps Runtime, VDS, and Aurelean through governed evidence.

Deliverables: Integration status docs, runtime main status report, downstream status reports.

Dependencies: Dry-run runtime stack.

Exit Criteria: Integration PRs merged and validated.

## Milestone 05: Portfolio OS V1 Certification

Objective: Certify the portfolio operating system for governed dry-run operation.

Deliverables: Certification report, inventory, readiness matrix, risk register, roadmap, validator.

Dependencies: Portfolio integration complete.

Exit Criteria: Certification validator passes and local certification commits exist.

## Milestone 06: VDS Runtime Pilot

Objective: Run a governed dry-run VDS Project Review Workflow.

Deliverables: Pilot charter, dry-run task set, audit package, owner review notes.

Dependencies: Portfolio OS V1 certification.

Exit Criteria: Dry-run pilot evidence complete; no production actions performed.

## Milestone 07: Aurelean Runtime Pilot

Objective: Run a governed dry-run Aurelean RFQ Review Workflow.

Deliverables: RFQ dry-run task set, supplier comparison evidence, audit package.

Dependencies: Portfolio OS V1 certification.

Exit Criteria: Dry-run pilot evidence complete; no vendor awards or external communications.

## Milestone 08: Controlled Execution Pilot

Objective: Design and test tightly scoped controlled execution gates.

Deliverables: Owner approval contracts, allowlists, audit logs, rollback plan, kill switch.

Dependencies: Successful dry-run pilots and owner authorization.

Exit Criteria: Controlled execution design validated; live execution still separately gated.

## Milestone 09: Production Readiness Certification

Objective: Certify whether production workflows can be considered for controlled execution.

Deliverables: Production readiness report, security review, operational runbooks, rollback evidence.

Dependencies: Controlled execution pilot results.

Exit Criteria: Owner-approved go/no-go decision.

## Milestone 10: Portfolio OS V2

Objective: Define the next-generation operating system for governed portfolio execution.

Deliverables: V2 architecture, governance model, runtime plan, certification criteria.

Dependencies: Production readiness certification.

Exit Criteria: V2 roadmap accepted by owner.
