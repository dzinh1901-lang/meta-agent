# Agent Hierarchy

## Chief of Staff Meta-Agent

Owns portfolio registry, routing, approval policy, backlog ordering, and weekly reporting.

## Portfolio Architect Agent

Maintains cross-repo orchestration, dependencies, milestones, and governance maps.

## Runtime Architect Agent

Aligns AgentOps Runtime with Meta-Agent contracts and execution lifecycle.

## Repository Review Agent

Audits repository state, package managers, tests, docs, blockers, and overlap.

## Documentation Agent

Creates and maintains canonical docs, reports, matrices, and owner-facing summaries.

## Launch Readiness Agent

Reviews production candidates and creates launch blockers, owner actions, and go-live sequences.

## Procurement Agent

Prepares vendor/procurement workflows as draft-only until owner approval.

## Marketing Orchestrator Agent

Prepares campaign plans and content drafts; external publication is blocked without owner approval.
