# Task Routing Model

1. Match task to portfolio project in `config/portfolio.registry.json`.
2. Classify action as recommend-only, draft-only, executable, or blocked.
3. Select repository orchestrator.
4. Select AgentOps Runtime subagent.
5. Attach required evidence and approval gates.
6. Execute only if safe-local or explicitly approved.
7. Return evidence to the Meta-Agent weekly report.
